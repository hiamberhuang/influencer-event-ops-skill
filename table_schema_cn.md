# 表格字段说明（中文）

## 01 WBS 总控

| 字段 | 说明 |
|---|---|
| 阶段 | 项目阶段，如立项、Rundown、场地、名单、预算、现场 |
| 工作包 | 一组相关任务 |
| 任务 | 可执行动作 |
| 输出物/检查标准 | 做到什么程度算完成 |
| 开始日期 | 任务开始时间 |
| DDL | 截止时间 |
| 负责人 | 唯一最终责任人 |
| 协作方 | 其他参与方 |
| 优先级 | P0/P1/P2 |
| 状态 | Not Started/In Progress/Blocked/Done |
| 风险/备注 | 卡点、依赖、风险 |

## 02 Rundown

每个环节必须回答：为什么存在、现场发生什么、谁负责。

## 03 场地动线

必须包含供应商报价和实地勘场结论。不要只看供应商照片。

## 04 达人名单邀请

必须分轮次：初步名单、第一轮邀请、第二轮邀请及补位、正式邀请函。

## 05 预算

预算应包含计划、预测、实际和差异。新增需求必须有 owner 和预算来源。

## 06 分工彩排

每个现场动作都要有 owner 和 backup。彩排必须真的走位。

## 07 座位表

座位号建议打通桌卡、手环、抽奖编号、乘车卡和签到表。
